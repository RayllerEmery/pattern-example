# Builder Example / Exemplo Builder

## PT-BR

Este pacote contem um exemplo simples do padrao Builder aplicado a criacao de objetos `User`.

### Arquivos

- `UserWithoutBuilder.java`: criacao por construtor com muitos parametros.
- `UserWithBuilder.java`: criacao com Builder em etapas.
- `ExecuteBuilder.java`: runner de demonstracao.

### Ideia principal

O Builder deixa a criacao mais legivel, evita parametros opcionais como `null` e melhora a manutencao quando o objeto cresce.

---

## EN

This package contains a simple Builder pattern example for creating `User` objects.

### Files

- `UserWithoutBuilder.java`: constructor-based creation with many parameters.
- `UserWithBuilder.java`: step-by-step creation with Builder.
- `ExecuteBuilder.java`: demonstration runner.

### Main idea

Builder makes object construction more readable, avoids optional `null` arguments, and improves maintainability as the object grows.
